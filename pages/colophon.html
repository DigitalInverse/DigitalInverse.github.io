Colophon

This blog is generated by Pelican, which takes great advantage of Python. The design is a minor twist on the zurb-F5-basic Pelican theme. The content is written in Markdown. Mathematical notation is expressed in LaTeX markup and rendered by the MathJax plugin. Source code is highlighted by Pygments, using a slightly modified version of Solarized Light. Blog comments are handled with the help of Disqus.
